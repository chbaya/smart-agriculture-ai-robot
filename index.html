AI Smart Agriculture Robot

This project presents an AI-based robot for smart irrigation and automatic weed detection. It combines Artificial Intelligence, IoT, and Robotics for sustainable farming.

Problem

  • Over-watering wastes water and damages crops.
  • Under-watering causes plant stress and low yield.
  • Manual weed removal is slow and costly.
  • Excessive herbicide harms crops and soil.

Solution

The robot waters plants only when needed and removes weeds automatically using AI vision. Data is sent to a dashboard via IoT.

Artificial Intelligence

YOLO and OpenCV run on Raspberry Pi for real-time weed detection. When a weed is detected, the Arduino activates the spraying mechanism.

How It Works

1. Robot moves along crop rows.
2. Sensors read soil and environmental conditions.
3. ESP32-CAM captures images.
4. Raspberry Pi AI detects weeds.
5. Arduino controls irrigation and spraying.
6. IoT dashboard shows live data.
